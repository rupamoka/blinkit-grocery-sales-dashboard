# BlinkIT Grocery Sales Dashboard

## Project Overview
Developed an interactive Power BI dashboard to analyze 
BlinkIT grocery sales data, outlet performance, product 
categories, and customer ratings.

## Tools Used
- Power BI
- Excel
- Data Visualization

## Dashboard Features
- KPI Cards
- Sales Trend Analysis
- Outlet Performance
- Product Category Analysis
- Interactive Filters/Slicers
